# 1.时间

用法1 ：
- 表示经常性或习惯性的动作或存在的状态。
- 表示 客观事实或普遍真理。
- 在时间、条件等状语从句中，用一般现在时表示将来。

用法2. 常用于表示频率的时间状语连用，如下：

```shell
always   // 永远
usually // 通常
every morning/night/evening/day/week/year  // 每....时候
often   // 常常
sometimes  // 有时候
occasionally  //  偶尔
from time to time  // 时不时
twice a week  // 每周两次
rarely   // 罕见
seldom  // 很少
once a month  // 一个月一次
hardly   // 几乎不
ever   // 从来..., 曾经...
never  // 决不
```

|一般过去时| Ved | 动词加ed | 
|现在完成进行时| have been + Ving | have been + 动词加ing|

|将来完成时| will + have + Ved | will + have +动词加ed |

## 1. 一般过去时 Ved
示例：
- What did you say just now?
重点：
```shell
常用过去时间状语连用 
如：yesterday（昨天），
two days ago（两天前），
last year（去年），
the other day（前几天），
once upon a time（过去曾经），
just now（刚才），
in the old days（过去的日子里）
```
### 1.1 过去进行时 was/were+Ving

示例：
- The report said the UFO was traveling east to west when he saw it 

### 1.2 过去完成时 had+Ved
示例：
- No sooner had we arrived home than it started to rain
  
重点：
- 为点动作： 一到家就下雨；it started to rain 
### 2.3 过去将来时 would+V
示例：
- He said he would stay with us.
  
重点：
- 一般现在时可替代过去将来时


## 1.2. 一般现在时 动词原型或动词加复数V/Vs
示例：
- We'll leave as soon as it stops raining
  
重点：
- as 程度副词； 一停雨，我们就走
### 1.2.1 现在进行时 am/is/are+Ving
示例：
- They are cleaning the house.
重点：  
- 现在进行时的肯定句
### 1.2.2 现在完成时 have+Ved

示例：
Have you done it?

重点：
- 现在完成时疑问句

### 1.2.3 现在完成进行时 have been Ving
示例：
- I have been looking forward to meeting you ever since we parted 

重点：
- 段动作：自从我们分开...然后...  ever since we parted
## 1.3. 一般将来时 will/shall +V 动词原形
示例：
- He will win in the competition. 

重点：
- 用于判断将来会发生的事情：competition 比赛 
### 1.3.1 将来进行时 will/shall+be +Ving
示例：
- I don't know what Jeanie will be doing at the time  tomorrow 

重点：
- 段动作：明天这个时候....；this time tomorrow
### 1.3.2 将来完成时 will + have+Ved

示例：
- I need the job badly,as my money will have run out by the end of next month 
  
重点：
- the end of
